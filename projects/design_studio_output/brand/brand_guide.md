## 1. Company Overview

**NovaTech** is a fictional technology company specializing in developer tools, cloud infrastructure, and enterprise SaaS platforms. The brand communicates reliability, innovation, and approachability.

**Tagline:** *"Engineering the future, shipping today."*

**Brand Personality:**
- Confident but not arrogant
- Technical but not inaccessible
- Modern but not trendy
- Professional but not stiff

---

## 2. Logo Usage

### Primary Logo
- The NovaTech wordmark uses Inter in bold weight
- The icon mark is a geometric hexagonal shape with an inner "N" cutout
- Always maintain minimum clear space equal to the height of the "N"
- Minimum size: 24px height for digital use

### Usage Rules
- **DO** use on white or neutral backgrounds
- **DO** use the all-white version on dark backgrounds (#0A0A14)
- **DON'T** stretch, rotate, or distort the logo
- **DON'T** add drop shadows, gradients, or outlines to the logo
- **DON'T** place on busy backgrounds without sufficient contrast
- **DON'T** change the logo color to anything other than #4F6FFF, #111827, or white

### Variants
| Variant | Use Case |
|---------|----------|
| Full logo (icon + wordmark) | Headers, hero sections, official documents |
| Icon only | Favicons, app icons, social avatars (min 32px) |
| Wordmark only | Inline text references, co-branding layouts |

## 4. Typography System

### Primary Typeface: Inter
Used for all UI text, body copy, headings, and navigation.

### Display Typeface: Playfair Display
Reserved for marketing hero sections, editorial content, and brand storytelling.

### Monospace Typeface: JetBrains Mono
Used for code snippets, API examples, terminal output, and technical data.

## 5. Tone and Voice

### Voice Characteristics
- **Direct:** Say what you mean. Use short, active sentences.
- **Helpful:** Always guide the user to the next step.
- **Technical but not exclusionary:** Use precise terminology but define it on first use.
- **Confident:** Use declarative statements. Avoid hedging language.

### Writing Guidelines
- Use contractions naturally ("you'll", "we've", "don't")
- Start with action verbs ("Build your first pipeline")
- Use the second person ("you", "your")
- Keep sentences to 1-2 ideas maximum
- Avoid buzzwords ("synergy", "paradigm shift", "best-in-class")
- Never apologize or hedge ("Unfortunately, this might not work...")

## 6. Photography Style

- Natural, diffused lighting. Avoid harsh flash.
- Rule of thirds composition with negative space for text overlays.
- Slight blue-cool color grade to align with NovaTech brand colors.
- Real people, real environments. Avoid stock-photo cliches.
- When showing code or interfaces, use actual product screenshots.

---

## 7. Icon Style

- Built on a 24x24 grid with geometric primitives
- 2px stroke width with rounded caps and joins
- Corner radius of 2px on rectangular elements
- Outlined icons for navigation, filled icons for primary CTAs and notifications
- Geometric, minimalist, slightly rounded style
