### Elements
- Workspace selector dropdown, email input, password input with visibility toggle
- Remember device checkbox, primary Sign In button (full width, 48px)
- OAuth buttons: GitHub, Google, SSO

### Interaction Flows
- Invalid credentials: inline error with red border
- Forgot password: opens /auth/reset modal
- Auto-redirect if already authenticated

## 3. User Settings

### Tabs: Profile | Notifications | Security | Billing

**Profile:** avatar upload, account details (name, email, role, timezone)
**Notifications:** email preferences, in-app toggles, digest frequency, muted topics
**Security:** password change, 2FA, active sessions, API access management
**Billing:** current plan, payment method, billing history, subscription actions

## 4. Data Table

### Layout
```
┌─────────────────────────────────────────────────────┐
│ Projects                                  [+ New]    │
│ [Status ▼] [Type ▼] [Date Range ▼] [🔍 Search]      │
│ Active(3)  Archived(2)  Draft(1)  All(6)            │
│                                                      │
│ [☐] │ Name              │ Status  │ Owner │ Action  │
│ [☐] │ Website Redesign  │ Active  │ Sam   │ [⋮]     │
│ [☐] │ API Migration     │ Active  │ Alex  │ [⋮]     │
│ [█] │ Auth Refactor     │ Active  │ Alex  │ [⋮]     │
│                                                      │
│ [Bulk: Archive | Delete | Transfer]                  │
│ Showing 1-6 of 42    [< 1  2  3  >]                 │
└─────────────────────────────────────────────────────┘
```

### Features
- Multi-select with bulk actions
- Filter bar with status pills, type dropdown, date range, search
- Sortable columns, resizable
- Row states: default, hover, selected, dragged
- 3-dot row actions: View, Edit, Duplicate, Archive, Delete
- Sticky pagination
