# Especialista: Frontend Dev
**ID:** `frontend_dev`
**Department:** Desenvolvimento Web / Frontend Development
**Arquivo:** `plugins/specialists/frontend_dev.py`

## Descricao

Desenvolvedor frontend especialista em tecnologias modernas — React, Vue, Angular, TypeScript, TailwindCSS, acessibilidade e performance.

## Arquitetura

- **Tipo de Hook:** PRE_LAUNCH, CONTEXT
- **Variavel de Ambiente:** `MYC_PLUGIN_FRONTEND_DEV=active`

### Contexto Injetado

- **Frameworks JS:** React (hooks, context, suspense), Vue 3 (Composition API, Pinia), Angular (servicos, RxJS)
- **Type avancado:** Genericos, utility types, type guards, discriminant unions, module augmentation
- **CSS moderno:** TailwindCSS, CSS Modules, Styled Components, SASS/LESS
- **Design responsivo:** Grid, Flexbox, container queries, mobile-first
- **Acessibilidade:** WCAG 2.2 AA, semantica HTML, ARIA labels, navegacao por teclado, screen readers
- **Performance:** Code splitting, lazy loading, tree shaking, image optimization, Core Web Vitals
- **Gerenciamento de estado:** Redux Toolkit, Zustand, Jotai, Vuex/Pinia, NGRX
- **Testes:** Jest, React Testing Library, Vitest, Cypress, Playwright
- **Build tools:** Vite, Webpack, esbuild, SWC
- **Padroes de componentes:** Composicao, compound components, render props, custom hooks, HOCs

myc agent add-plugin meu_agente frontend_dev
